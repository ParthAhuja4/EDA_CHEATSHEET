ROUTER

Answer these 4 questions first

This tells you exactly which phases and blocks apply to your dataset.

Q1. What is your target variable?
   ├── Continuous number (price, sales, temp)  → REGRESSION track
   ├── Binary (yes/no, fraud/not, 0/1)      → BINARY CLASSIFICATION track
   ├── 3+ categories (dog/cat/bird)        → MULTICLASS track
   └── No target                          → UNSUPERVISED / PURE EDA track

Q2. Does the dataset have a meaningful date/time column?
   └── Yes → add TIME SERIES BLOCK (Phase 6)

Q3. Does the dataset have free-text columns?
   └── Yes → add TEXT BLOCK (Phase 7)

Q4. How large is the dataset?
   ├── < 10k rows   → load fully, use all plots freely
   ├── 10k–1M rows → sample for plots (50k max), full data for stats
   └── > 1M rows   → always sample for EDA, use chunking / Dask

Outlier check (IQR method)

for col in continuous_features:
    Q1, Q3 = df[col].quantile(0.25), df[col].quantile(0.75)
    IQR = Q3 - Q1
    lower, upper = Q1 - 1.5*IQR, Q3 + 1.5*IQR
    n_out = df[(df[col] < lower) | (df[col] > upper)].shape[0]
    print(f"{col}: {n_out} outliers ({n_out/len(df)*100:.1f}%)")

PHASE 4

Feature vs Target — Bivariate

Find which features actually carry signal for prediction.

4A — Categorical feature vs Binary/Multiclass target

Statistical test: Chi-Square

from scipy.stats import chi2_contingency
for col in categorical_features:
    if col == 'target': continue
    p = chi2_contingency(pd.crosstab(df['target'], df[col]))[1]
    result = "SIGNIFICANT ✓" if p < 0.05 else "not significant ✗"
    print(f"{col:30s}  p={p:.4f}  {result}")

Graphs — use all three together:

# 1. Raw count split — shows volume
sns.countplot(x=col, hue='target', data=df)

# 2. Rate — more honest when category sizes differ
(100 * df[df['target']==pos_class][col].value_counts()
 / df[col].value_counts()).sort_values().plot(kind='bar')

# 3. Exact proportions table
df.groupby(col)['target'].value_counts(normalize=True).to_frame() * 100
→Why all three: count plot shows volume, rate chart shows success rate. A rare category can have 90% success rate — you'd miss that with count plot alone.

4B — Categorical feature vs Continuous target

Statistical tests: ANOVA or Kruskal-Wallis

from scipy.stats import f_oneway, kruskal
for col in categorical_features:
    groups = [df[df[col]==c]['target'].dropna() for c in df[col].unique()]
    p_anova   = f_oneway(*groups).pvalue
    p_kruskal = kruskal(*groups).pvalue
    print(f"{col:30s}  ANOVA p={p_anova:.4f}  Kruskal p={p_kruskal:.4f}")

# Graphs
sns.boxplot(x=col, y='target', data=df)
sns.violinplot(x=col, y='target', data=df)
df.groupby(col)['target'].mean().plot(kind='bar')

4C — Continuous feature vs Binary target

Statistical tests: KS Test or t-test

from scipy.stats import ks_2samp, ttest_ind
groups = df['target'].unique()
for col in continuous_features:
    g = [df[df['target']==c][col].dropna() for c in groups]
    p_ks = ks_2samp(g[0], g[1]).pvalue
    p_t  = ttest_ind(g[0], g[1]).pvalue
    print(f"{col:30s}  KS p={p_ks:.4f}  t-test p={p_t:.4f}")

# Graphs
sns.histplot(data=df, x=col, hue='target', kde=True, multiple='stack')
sns.boxplot(data=df, x='target', y=col)
sns.violinplot(data=df, x='target', y=col)

PHASE 5

Feature vs Feature — Multivariate

Find redundant features and unexpected interactions.

5A — Numeric vs Numeric (multicollinearity)

corr = df[numeric_features].corr()
sns.heatmap(corr, annot=True, fmt='.2f', cmap='coolwarm', center=0)

# Flag dangerous pairs
import numpy as np
upper = corr.where(np.triu(np.ones(corr.shape), k=1).astype(bool))
high_corr = [(col, row, upper.loc[row, col])
             for col in upper.columns for row in upper.index
             if abs(upper.loc[row, col]) > 0.7]
print("High correlation pairs (|r| > 0.7):", high_corr)
→If |r| > 0.7 between two features: keep one, drop or transform the other for linear models. Tree-based models are less sensitive to multicollinearity.

5B — Categorical vs Categorical (Cramér's V)

from scipy.stats import chi2_contingency
def cramers_v(x, y):
    cm  = pd.crosstab(x, y)
    chi2 = chi2_contingency(cm)[0]
    n    = cm.sum().sum()
    r, k = cm.shape
    return np.sqrt(chi2 / (n * (min(r, k) - 1)))

for col in categorical_features:
    for col2 in categorical_features:
        if col >= col2: continue
        v = cramers_v(df[col], df[col2])
        if v > 0.6:
            print(f"{col} vs {col2}: V={v:.2f} — HIGH ASSOCIATION")

5C — Numeric vs Categorical
